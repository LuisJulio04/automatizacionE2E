Segundo ejercicio dentro de las opciones realizado en y almacenado en éste repositorio.



Instrucciones de ejecución:

1. Descarga el repositorio en tu computadora local. (https://github.com/LuisJulio04/automatizacionE2E)
2. Abre el entorno de ejecución de pruebas (el Test Runner de Cypress).
3. Importa la carpeta "e2e" que acabas de descargar como parte de tu proyecto de pruebas automatizadas.
4. Selecciona la opción "E2E Testing" para ejecutar las pruebas de extremo a extremo.
5. En la ventana que se abre, elige el navegador de tu preferencia entre las opciones disponibles.
6. Haz clic en "Start E2E Testing" o en el botón de inicio correspondiente para iniciar la ejecución de las pruebas.
7. En la nueva ventana del navegador que se abre, busca y selecciona el archivo "spec.cy.js" dentro de la categoría "Specs".
8. Observa cómo las pruebas se ejecutan paso a paso y verifica que cumplen con las especificaciones establecidas.

Siguiendo estos pasos, podrás ejecutar las pruebas de extremo a extremo utilizando el archivo de especificación proporcionado y verificar que se ejecuten correctamente de acuerdo con las especificaciones establecidas.




Especificaciones otorgadas para éste test:

Realizar una prueba funcional automatizada (Prueba E2E) de un flujo de compra en la página https://www.saucedemo.com que incluya:
1. Autenticarse con el usuario: standard_user y pasword: secret_sauce
2. Agregar dos productos al carrito
3. Visualizar el carrito
4. Completar el formulario de compra
5. Finalizar la compra hasta la confirmación "THANK YOY FOR YOUR ORDER"
